            Family Connection Tree is a private, easy-to –use website where you can connect with your friends and family.

This is how it happened...


  • At first it was going to just be a way to list relationships like spouse, and kids. 

  • But that wasn’t good enough, because it wouldn’t link the users together, and wouldn’t add much value

  • Then decided to just do a full blown genealogy system it was overkill, and would take more time do.


This is what you will get...


  • It is a hybrid approach. 

  • A very lightweight genealogy system, that you could view the relationship list form on your profile.

  • This system include many things like and this will be used more for showing relationships.


Modules..

Authentication and Profile Contact Book News Memories Facts

Authentication and Profile : Every user wil have their own login portal. They are given to signup if already not a user. They have to update their profile an include information such as Email,Phone number, Profile picture, Date of birth etc

Contact : A user can view all the contact information such as email and phone numbers of their family members.

News : Here, the user gets to see all the exciting news ad upcoming events of their families. Create a new News post by clicking add new .

Memories : All the memories like trips, birthdays and anniversary is all available here. Create a new memory post by clicking add new .

Facts : All the facts and trivias about the family is visible here. Create a new Fact post by clicking add new.
